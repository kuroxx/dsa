# Trees

## Tree Traversal

| Depth first | Breadth first |
|---|---|
| 1. Pre-order  | 1. Level-order | 
| 2. In-order   | | 
| 3. Post-order | | 

### Depth first

1. Pre-order O(n)
    - Visit node
    - Traverse left
    - Traverse right

2. In-order O(n)
    - Traverse left
    - Vivist node
    - Traverse right

3. Post-order O(n)
    - Traverse left
    - Traverse right
    - Visit node

### Breadth first

1. Level-order O(n)
    -  Traverse level by level, left to right
